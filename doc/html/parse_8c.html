void parseDoc ( const char *  docname,
osm *  map 
)

This function parses an osm file.

parseDoc parses the file docname using xmlParseFile then tests if parsing is successful, if the obtained tree is not empty and if its is named 'osm'. parseDoc then counts the number of 'node', 'way' and 'relation' nodes and allocates map vectors to the appropriate size. parseDoc then fills the fields in the map structure by calling the appropriate parsing functions.
